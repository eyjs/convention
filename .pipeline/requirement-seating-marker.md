# 요구사항: 좌석 배치도 — 테이블 마커 시스템으로 전환

## 핵심 변경
기존 핀(1인 1개) → **테이블 마커(1테이블 N명)** + **엑셀 배정**

## 관리자 에디터

### UI
```
┌──────────────────────────────────────┐
│ ← 레이아웃명   [💾저장] [📥다운로드] [📤업로드] │
├──────────────────────────────────────┤
│                                      │
│  배경 이미지 (도면)                    │
│                                      │
│     ①    ②    ③                     │  ← 번호 붙은 테이블 마커
│                                      │
│     ④    ⑤    ⑥                     │
│                                      │
│  [📷배경] [➕테이블추가] [🔒잠금]       │
├──────────────────────────────────────┤
│ 테이블 1 (4명) | 테이블 2 (6명) | ... │  ← 하단 테이블 목록
└──────────────────────────────────────┘
```

### 동작
1. **배경 이미지 업로드** (도면)
2. **테이블 추가**: 클릭 → 원형/사각형 마커 생성 (번호 자동 부여)
3. **마커 드래그**: 위치 이동
4. **마커 리사이즈**: 크기 조절 (원형 반지름, 사각 너비/높이)
5. **번호 수정**: 마커 더블클릭 또는 우측 패널에서
6. **참석자 다운로드**: 행사 참석자 + 테이블번호 컬럼 엑셀 생성
7. **참석자 업로드**: 엑셀의 테이블번호 → members 자동 매핑
8. **자동 저장** + 명시적 저장 버튼

### 엑셀 다운로드 형식
| 번호 | 이름 | 전화번호 | 테이블번호 |
|---|---|---|---|
| 1 | 홍길동 | 010-1234-5678 | 1 |
| 2 | 김영희 | 010-2345-6789 | 1 |
| 3 | 이철수 | 010-3456-7890 | |

- 이미 배정된 사람은 테이블번호 채워짐
- 미배정은 빈칸
- 재다운로드 시 현재 상태 반영

### 엑셀 업로드
- 테이블번호 컬럼 값으로 members 매핑
- 이름+전화번호로 기존 참석자 매칭
- 없는 테이블번호 → 경고
- 결과: "30개 테이블에 295명 배정 완료, 5명 미배정"

## 데이터 구조 (LayoutJson)

```json
{
  "tables": [
    {
      "id": "t1",
      "number": "1",
      "label": "1번 테이블",
      "shape": "circle",
      "x": 200, "y": 300,
      "width": 80, "height": 80,
      "color": "#3b82f6",
      "members": [
        { "userId": 1, "name": "홍길동" },
        { "userId": 2, "name": "김영희" }
      ]
    }
  ]
}
```

## 사용자 뷰

```
┌───────────────────┐
│ ← 내 자리          │
├───────────────────┤
│                   │
│  도면 이미지       │
│  + 테이블 마커     │
│                   │
│  ①  ②  ⑳ ← 내것 │
│                   │
│  줌+핀치/터치팬    │
├───────────────────┤
│ 📍 20번 테이블     │
│ 홍길동, 김영희     │
│ 이철수, 박지성     │
│        [내자리로]  │
└───────────────────┘
```

- 내 테이블 빨간 강조 + 바운스
- 다른 테이블 파란색 + 번호
- 테이블 탭 → 멤버 목록 모달
- 하단에 내 테이블 정보 + 같은 테이블 사람들

## API 변경

### 신규
| Method | Endpoint | 설명 |
|---|---|---|
| GET | `/api/admin/seating-layouts/{id}/members/download` | 참석자+테이블번호 엑셀 다운로드 |
| POST | `/api/admin/seating-layouts/{id}/members/upload` | 엑셀 업로드 → 테이블 배정 |

### 기존 유지
- CRUD API 그대로 (LayoutJson 구조만 변경)
- 사용자 API 그대로

## 알림톡 변수
`#{table_number}` → 해당 사용자의 테이블 번호

## 모바일 현장 편집 (MUST)

### 시나리오
```
관리자 모바일 → 좌석 배치도 → 테이블 20번 탭
→ 멤버: 홍길동, 김영희, 이철수, 박지성
→ 홍길동 탭 → "이동할 테이블 선택" → 3번 선택
→ 홍길동이 20번에서 사라지고 3번에 추가됨
→ 자동 저장
```

### UI (모바일)
```
┌───────────────────┐
│ 20번 테이블 (4명)   │
├───────────────────┤
│ 홍길동     [→ 이동] │
│ 김영희     [→ 이동] │
│ 이철수     [→ 이동] │
│ 박지성     [→ 이동] │
├───────────────────┤
│ [+ 사람 추가]       │
└───────────────────┘

"이동" 탭 시:
┌───────────────────┐
│ 홍길동 → 어디로?    │
├───────────────────┤
│ 1번 (4명)          │
│ 2번 (6명)          │
│ 3번 (5명)  ← 선택  │
│ ...                │
└───────────────────┘
```

### 동작
- 테이블 마커 탭 → 멤버 목록 모달
- 멤버 옆 "이동" 버튼 → 테이블 선택 모달 → 즉시 이동 + 자동 저장
- "사람 추가" → 미배정 참석자 검색 → 배정
- 모든 변경 즉시 저장 (API PUT)
